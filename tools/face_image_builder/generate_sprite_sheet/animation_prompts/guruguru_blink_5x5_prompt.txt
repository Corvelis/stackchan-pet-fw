添付した Image A のキャラクターを元に、顔向き追従アバター用の5列×5行リファレンスシートを1枚作成してください。
Image A はキャラクターの同一性と正面基準の参照元です。

この画像の目的は、同じキャラクターの頭部を25方向に並べた、顔向き追従アバター用の5×5方向差分シートを作ることです。
25コマは別々のイラストではなく、同じキャラクターの同じ頭部を、向きだけ変えて並べたリファレンスシートにしてください。
中央セル（3行3列目）は Image A の正面画像にできるだけ忠実に一致させてください。

最優先レイアウト要件：
- 出力は1枚の正方形画像にしてください
- 画像全体を、5列×5行の完全に均等なグリッドとして構成してください
- 合計25コマを必ず配置してください
- 各セルは完全に同じサイズにしてください
- 各列の幅は完全に同じにしてください
- 各行の高さは完全に同じにしてください
- 各セルの間隔を均等にしてください
- 1セルにつき1人だけ配置してください
- 特に5行目（一番下の段）も、1〜4行目と完全に同じ高さ、同じ余白、同じ比率にしてください
- 一番下の段だけ縦に圧縮しないでください
- 一番下の段だけ人物を小さくしないでください
- 一番下の段だけ上下余白を変えないでください
- 一番下の段の人物が画像の下端に近づきすぎないようにし、他の行と同じだけ下側に黒い余白を残してください
- 枠線、グリッド線、文字、番号、説明、矢印、装飾は入れないでください

キャラクター統一：
- 25コマすべて同じキャラクターにしてください
- 同じ絵柄、同じ髪型、同じ髪色、同じ目の形、同じ服装、同じアクセサリー、同じ配色にしてください
- 顔の輪郭、前髪、髪型、アクセサリー、衣装の襟や模様、全体の比率をできるだけ変えないでください
- 別人のように描き直さないでください
- 中央セル（3行3列目）は、Image A の正面画像に極力一致させてください
- 中央セルを同一性の基準画像として扱い、周囲24コマはその正面画像から自然に向きだけ変化したバリエーションにしてください

変えてよい要素：
- 各コマで変えてよいのは、頭の向きと視線だけです
- それ以外の要素は可能な限り固定してください
- 表情は全コマでニュートラルにしてください
- 目は閉じた状態にしてください
- 口は閉じた状態にしてください

向きの配置：
- 横方向は、完成画像を見る人から見て、左向き → 左斜め → 正面 → 右斜め → 右向き
- 縦方向は、上向き → やや上 → 正面 → やや下 → 下向き
- 中央が正面です
- 左から右へ、顔の向きが連続的に変化するようにしてください
- 上から下へ、頭の向きと視線が連続的に変化するようにしてください

左右方向の指定：
- 1列目：顔と鼻先が画像の左端方向を向いている
- 2列目：顔と鼻先が少し画像の左端方向を向いている
- 3列目：正面を向いている
- 4列目：顔と鼻先が少し画像の右端方向を向いている
- 5列目：顔と鼻先が画像の右端方向を向いている

縦方向の指定：
- 1行目：顔と鼻先が上方向を向いている
- 2行目：顔と鼻先がやや上方向を向いている
- 3行目：正面の高さを向いている
- 4行目：顔と鼻先がやや下方向を向いている
- 5行目：顔と鼻先が下方向を向いている

見た目のルール：
- 顔の向きは、視線だけでなく、輪郭、鼻先、顎、耳、前髪、アクセサリー、首の見え方が自然に変化することで表現してください
- 左右方向は、完成画像を見る人から見た左右で判断してください
- キャラクター本人から見た左右ではなく、完成画像上で鼻先が向く方向を基準にしてください
- 左右反転したコピーではなく、同じ頭部が自然に回転しているように見せてください
- キャラクターデザイン自体は変えないでください

構図：
- 各セル内でキャラクターを中央寄せにしてください
- 胸上のバストアップ構図にしてください
- 顔の大きさ、顔の位置、頭頂の位置、目の高さ、顎の位置、肩の位置、構図、トリミングをできるだけ統一してください
- 各セルの中でキャラクターが収まる範囲を統一してください
- 顔が上向きや下向きになっても、人物の大きさや上下方向の余白を変えないでください
- 下向きの顔になっても、頭部や肩を下にずらさず、各セル内の中央位置を保ってください
- 髪や肩が隣のセルにはみ出しすぎないようにしてください
- 25コマ全体が、1人のキャラクターの顔向き差分シートとして自然に見えるようにしてください

背景：
- 背景は全25コマで完全に統一された純黒の単色背景にしてください
- 背景はフラットな黒一色にしてください
- グラデーション、模様、影、発光、装飾、風景、テクスチャは入れないでください
